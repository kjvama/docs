Purpose
==================================================
English Version
This document defines the document numbering system, classification rules, and documentation standards for the KJVAMA project.

Korean Version
본 문서는 KJVAMA 프로젝트에서 사용하는 문서 번호 체계, 분류 규칙 및 문서 작성 기준을 정의한다.

==================================================
Scope
==================================================
This document applies to:

- Development documents
- Project management documents
- Content production documents
- Bible knowledge documents

This document does not define the contents of individual documents.

Document Template
==================================================
English Version
By default, all documents follow the standard structure shown below. Individual documents may add or omit sections as necessary.

- History
- Purpose
- Background
- Scope
- Principles
- Decision
- Reference
- Approval
